Segundo Diego Eis, no seu livro HTML5 e CSS3 com farinha e pimenta:

O HTML é baseado no conceito de Hipertexto, que é uma forma de organizar conteúdo de forma não linear. Hipertexto são conjuntos de elementos - ou nós - ligados por conexões. Estes elementos podem ser palavras, imagens, vídeos, áudio, documentos etc. Estes elementos conectados formam uma grande rede de informações. Eles não estão conectados linearmente como se fossem textos de um livro, onde um assunto é ligado ao outro seguidamente.
